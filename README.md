# Netflix Exploratory Data Analysis (EDA)

## Project Overview
Exploratory Data Analysis of Netflix Movies and TV Shows to uncover trends in content type, growth, genres, ratings, and durations.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Netflix Movies and TV Shows dataset containing information on content type, country, genre, rating, duration, and release year.

## Key Analysis
- Movies vs TV Shows distribution
- Content growth trends over time
- Country-wise and genre-wise analysis
- Rating and audience targeting insights
- Movie duration and TV show season analysis
- Missing value and data quality analysis

## Key Insights
- Netflix has more Movies than TV Shows.
- The number of titles added to Netflix increased rapidly after 2015.
- Most Netflix content comes from the USA and India.
- Most movies on Netflix are between 90 and 120 minutes long.
